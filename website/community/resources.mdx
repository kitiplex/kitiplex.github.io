## Help us improve

:::warning[Reminder]

[Kitiplex](https://kitiplex.github.io/) has adopted a Code of Conduct that we expect project participants to adhere to. We encourage you to read **[the full text](../community/code-of-conduct.mdx)** so that you can understand what actions will and will not be tolerated.
:::


We appreciate your effort in helping us improve. Prior to submitting your issues, run the following command to ensure all cache has been cleared:

```bash npm2yarn
npm run clear
```

Additionaly, consider the following:

* Is it evident that this is indeed an issue?

* Do you possess fundamental insights into the potential areas of malfunction?

* Could the problem be attributed to errors on your part?

If took some steps to reproduce, please provide the following:

1. In this enviroment...
2. With this configuration...
3. Run '...'
4. See error...

That's it!

File it [here](https://github.com/kitiplex/iuiu/issues/new/choose).
