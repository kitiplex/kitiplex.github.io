### Network

Here's an extended command that might be helpful in fix issues with network. **Note** that this requires elevated / admin access and must be execute in the exact order.

```sh
ipconfig /release
```
```sh
ipconfig /flushdns
```
```sh
ipconfig /renew
```
```sh
netsh int ip reset
```
```sh
netsh winsock reset
```
Restart comupter.

# Upgrades

### Check Licenses


* For OEM.

```sh
wmic path softwarelicensingservice get OA3xOriginalProductKey
```

* For Volumes. Check Host.
```sh
slmgr.vbs /dli
```
* Disable KMS host caching:
```sh
slmgr.vbs /ckhc
```
* Enable KMS host cahing:

```sh
slmgr.vbs /skhc
```
