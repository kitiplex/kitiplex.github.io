### For statements

```py
words = ["cat", "dog", "monkey"]

for w in words:
    print(w, len(w))
```
:::note[Output]
```md
cat 3
dog 3
monkey 6
```
:::

```py
a = ['Mary', 'had', 'a', 'little', 'lamb']

for i in range(len(a)):
    
    print(i, a[i])
```
:::note[Output]
```0 Mary
1 had
2 a
3 little
4 lamb
```
:::

```py
for n in range(2, 10):
    for x in range(2, n):
        if n % x == 0:
            print(n, 'equals', x, '*', n//x)
            break
    else:
        # loop fell through without finding a factor
        print(n, 'is a prime number')
```

:::note[Output]
```2 is a prime number
3 is a prime number
4 equals 2 * 2
5 is a prime number
6 equals 2 * 3
7 is a prime number
8 equals 2 * 4
9 equals 3 * 3
```
:::

### Array


```py
def print_triangle(height):
    for i in range(1, height + 1):
        print(' ' * (height - i) + '*' * (2 * i - 1))

# Example usage
print_triangle(5 * 2)
```
